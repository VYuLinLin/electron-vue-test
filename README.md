## Install and use

- Get the project code

```bash
git clone https://github.com/VYuLinLin/electron-vue-test.git
```

- Installation dependencies

```bash
cd electron-vue-test

npm install

```

- run

```bash
npm run start
```

- build

```bash
npm run build-e
```
